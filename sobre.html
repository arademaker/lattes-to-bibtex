Conversor Lattes-BibTeX

Esta ferramenta foi criada como produto secundário do projeto Semantic Lattes. Após desenvolvermos a transformação de Lattes para RDF, tivemos a idéia de que seria útil para boa parte da comunidade científica brasileira, ter um extrator das referências bibliográficas de um Lattes.

Como usar?

Para usar esta ferramenta, um pesquisador deverá seguir os seguintes passos:

  1. Acessar o site do Lattes escolhendo o link para atualizar o currículo e logar-se no sistema Lattes usando seus dados.
  2. Na aba de ferramentas inferior, na página inicial do Lattes, escolher a opção de exportação do Lattes. Na janela de exportação, escolher a opção XML e clicar em Confirmar.
  3. Seu navegador iniciará o download de um arquivo com nome CV_XXXXXXXXXXXXXXXX.zip onde XXX... é o código do seu Currículo Lattes.
  4. Você pode submeter para o o Lattes-BibTeX este arquivo ZIP, não precisa descompactar. O Lattes-BibTeX irá então gerar um BibTeX com todas as produções bibliográficas encontradas no seu Currículo Lattes.

Que dados são considerados no Lattes?

  • Teses de doutorado e dissertações de mestrado informadas pelo autor do Lattes em sua formação acadêmica.
  • Todas as produções bibliográficas: trabalhos em eventos, artigos publicados, capítulos em livros, livros e artigos aceitos para publicação.

Meus dados são salvos pelo Lattes-BibTeX?

Não. Seus dados não são salvos neste sistema. Os arquivos submetidos são transformados em BibTeX e descartados. Os usuários que mesmo assim preferirem remover dados pessoais dos seus Lattes antes de submeter o arquivo para este sistema podem fazê-lo, desde que o arquivo XML editado, antes de submetido, permaneça um XML Lattes válido. No XML do Lattes, a única tag que pode conter alguma informação sensível é a tag ENDERECO, caso o pesquisador tenha escolhido seu endereço residencial como endereço principal.
